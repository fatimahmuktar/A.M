تحويل ملفات الإكسل إلى قاعدة بيانات هو الخطوة الاحترافية والأهم لضمان استقرار مشروع "NEU AttendAI". بدلاً من الاعتماد على ملفات CSV يدوية، ستستخدم قاعدة بيانات تتحدث تلقائياً.
إليك أفضل استراتيجية تقنية للقيام بذلك:
### 1. اختيار قاعدة البيانات (السحابة هي الأفضل)
استخدم **Supabase** (وهي بديل مفتوح المصدر لـ Firebase). هي مثالية لأنها:
 * تتعامل مع البيانات كأنها "جدول إكسل" سهل القراءة.
 * تسمح لك برفع ملفات CSV بضغطة زر واحدة لتتحول مباشرة إلى جداول (Tables).
 * توفر "API" جاهزاً يربط موقعك بقاعدة البيانات فوراً.
### 2. دورة حياة البيانات (كيف تضمن التحديث كل فصل)
بما أن البيانات تتغير، لا تقم ببرمجة النظام ليعتمد على "ملف"، بل اجعل النظام يعتمد على "استعلام" (Query):
 * **الأدمن (Admin):** يقوم برفع ملف الإكسل الجديد إلى "لوحة تحكم الأدمن".
 * **المعالجة (Script):** يقوم كود برمجي (Node.js أو Python) بقراءة الملف وتحديث الجداول في Supabase (حذف البيانات القديمة أو إضافة فصل جديد).
 * **الموقع (Client):** عندما يفتح الطالب الموقع، يقوم الكود بطلب البيانات من Supabase. بذلك، بمجرد تحديث الأدمن للقاعدة، يرى الطالب البيانات الجديدة **فوراً** دون تغيير أي سطر كود.
### 3. البرومت الموحد لتنفيذ هذه النقلة النوعية
يمكنك استخدام هذا البرومت لمساعدك البرمجي لبناء هذا النظام:
> **"Help me migrate from static CSV files to a dynamic database using Supabase. Create an Admin Dashboard functionality where the admin can upload a new term's Excel/CSV file to automatically update the 'Courses' table in Supabase. Implement the frontend to fetch course data directly from the Supabase API instead of reading local files. Ensure the system handles data versioning so that old terms are archived and the new term's data is displayed for students and professors."**
> 
### نصيحة إضافية للهيكلية (Architecture):
لضمان أن يكون نظامك مرناً جداً لكل فصل دراسي:
 1. **عمود الفصل (Term Column):** أضف عموداً في جدول الكورسات باسم semester (مثلاً: Fall2026, Spring2026).
 2. **الفلترة:** اجعل الكود الخاص بالطالب يقوم بجلب الكورسات بناءً على semester الحالي فقط.
 3. **الأرشفة:** هذا يتيح لك الاحتفاظ بتاريخ الكورسات لسنوات دون حذفها، فقط عبر تغيير "الفصل النشط" (Active Semester) في إعدادات الأدمن.
هل تود أن أوضح لك كيفية ربط موقعك بـ Supabase برمجياً، أم تفضل التركيز على تصميم واجهة لوحة تحكم الأدمن أولاً؟
